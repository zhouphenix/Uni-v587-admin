# v587-todo

## 介绍
顾名思义， 此项目功能即：待处理事务; 初衷即是记录手中的事务。
在插件市场搜索了下， 发现没有此类插件， 为了方便工作， 就将vue中TodoMVC移植过来

## 使用
按组件的方式使用（easycom、uni_modules）

```vue
<v587-todo ></v587-todo>
```
